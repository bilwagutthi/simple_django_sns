## 3. Setup
1. Create venv
> python -m venv env
2. Switch to venv
> source env/Scripts/activate
3. Install Django
> pip install django
4. Create requirements file
> pip freeze > requirements.txt
5. Create Django Project
> django-admin startproject sns
